# 🌍 Task 03 - Geospatial Analysis

## 🎯 Objective
To visualize restaurant locations using latitude and longitude data and analyze their distribution across cities and countries.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Folium
- Google Colab

## 📌 Steps Performed
1. Loaded the dataset
2. Explored latitude and longitude data
3. Visualized restaurant locations on a map
4. Analyzed restaurant distribution across cities
5. Analyzed restaurant distribution across countries
6. Explored correlation between restaurant location and ratings

## 💡 Key Insights
- Restaurant locations were successfully visualized using maps  
- Some cities contain significantly more restaurants than others  
- Restaurant distribution varies across countries  
- Aggregate ratings show variation across geographical locations  
- Geospatial analysis helps understand restaurant concentration patterns
